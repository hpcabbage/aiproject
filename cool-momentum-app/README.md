# Momentum

一个偏酷感视觉的待办 + 习惯打卡手机 App，当前已完成到第二版 MVP。

## V2 已完成

- 今日待办列表：新增、完成、删除
- 习惯打卡：每日点击打卡 / 取消打卡
- 分类系统：专注 / 健康 / 生活 / 学习
- 主页分类筛选
- 焦点提示文案
- 节奏洞察卡片
- 统计页：完成率、最佳连续天数、总完成项
- 分类分布统计
- 近 7 天习惯打卡格子
- AsyncStorage 本地持久化
- Expo / React Native 跨平台项目骨架

## V2.1 刚补上

- 待办优先级：高 / 中 / 低
- 新增待办时可直接选择优先级
- 待办列表自动按“未完成优先 + 优先级 + 最新创建”排序
- 老数据自动补齐 priority 字段，不会把已有本地数据搞坏
- 待办 / 习惯支持直接编辑，不用删掉重建
- 编辑态支持直接删除
- 首页新增 `今日 Top 3` 焦点区域，强化每天最重要的推进项
- 统计页新增近 7 天 / 30 天趋势图
- 记录待办完成时间，后续趋势统计会越来越准
- 新增每日提醒 MVP：任务 / 习惯可单独开启提醒并设置时间
- 已接入 `expo-notifications` 与 `expo-device`
- 完成待办后会自动停掉对应提醒，避免完成了还一直催
- 提醒文案做了产品化处理，不再只是生硬系统提示
- 统计页新增高优先级完成率、分类执行洞察
- 首页验收视角已完成一轮交互收口：主路径层级稳定，Quick Add 已退回辅助入口；当前状态为“代码侧可收口、视觉侧待补最终验收”

## 启动

```bash
npm install
npm run start
```

然后用 Expo Go 扫码即可在手机上预览。

## 技术栈

- TypeScript
