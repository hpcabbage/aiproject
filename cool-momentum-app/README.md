# Momentum

一个偏酷感视觉的待办 + 习惯打卡手机 App MVP。

## 已完成

- 今日待办列表
- 习惯打卡
- 今日完成率
- 连续天数展示
- 统计页
- AsyncStorage 本地持久化
- Expo / React Native 跨平台项目骨架

## 启动

```bash
npm install
npm run start
```

然后用 Expo Go 扫码即可在手机上预览。

## 技术栈

- Expo
- React Native
- TypeScript
- AsyncStorage
