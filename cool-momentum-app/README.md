# Momentum

一个偏酷感视觉的待办 + 习惯打卡手机 App，当前已完成到第二版 MVP。

## V2 已完成

- 今日待办列表：新增、完成、删除
- 习惯打卡：每日点击打卡 / 取消打卡
- 分类系统：专注 / 健康 / 生活 / 学习
- 主页分类筛选
- 焦点提示文案
- 节奏洞察卡片
- 统计页：完成率、最佳连续天数、总完成项
- 分类分布统计
- 近 7 天习惯打卡格子
- AsyncStorage 本地持久化
- Expo / React Native 跨平台项目骨架

## 启动

```bash
npm install
npm run start
```

然后用 Expo Go 扫码即可在手机上预览。

## 技术栈

- Expo
- React Native
- TypeScript
- AsyncStorage
